# Описание проекта

daylist - это offline-first приложение для управления личными задачами, идеями и финансами

## Основные функциональные возможности

- список дел
- заметки
- дневник доходов и расходов

## Технологии

### Фронтенд

- Next.js,
- tailwindcss,
- typescript
- storybook
- shadcn/ui
- radix-ui
- indexeddb - для оффлайн-режима

### Бэкенд

- Nest.js,
- postgresql,
- typeorm

### Инфраструктура

- docker - контейнеризация
- github actions - CI/CD

## Цели проекта

- создать удобное и функциональное приложение для управления личными задачами, идеями и финансами, которое может
  работать без необходимости регистрации и авторизации в оффлайн-режиме, сохраняя данные на устройстве пользователя
- обеспечить синхронизацию данных между устройствами при наличии интернет-соединения и зарегистрированного аккаунта

## Требования к интерфейсу

### Общие

- адаптивный дизайн для различных устройств 320px, 768px, 1024px, 1600px
- максимальная ширина экрана - 1600px. При этой ширине и больше, контент центрирован, а по краям экрана остаются поля
- одна цветовая схема - что-то среднее между светлой и темной темой, спокойные тона с минимальной нагрузкой на зрение

### Список дел

- интуитивно понятный и простой в использовании интерфейс, drag-and-drop, возможность вкладывать подзадачи в задачи
  перетаскиванием
